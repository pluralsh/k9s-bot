# k9s-bot

controller for our k9s doggo robot

## Quick Start

Use `uv` to manage local builds of the project, in particular:

```ssh
uv venv
source .venv/bin/activate
git clone git@github.com:pluralsh/unitree_webrtc_connect.git
uv pip install -r requirements.txt
uv run doggo.py
```

To exit the venv, run `deactivate`
